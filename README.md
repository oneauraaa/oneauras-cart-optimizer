# oneaura's Cart Optimizer

## Overview

oneaura's Cart Optimizer is a mod that optimizes the handling of minecarts by removing the item client-side immediately upon placement. This prevents carts from ghosting (you place the cart, but it still appears to be on your hotbar).

## Commands
| Commands | Usage |
| :--- | :--- |
| /cartoptimizer on | Enable the mod. |
| /cartoptimizer off | Disable the mod. |
| /cartoptimizer debug <on/off> | Toggle the `[Cart Optimizer] -1 Cart` indicator. |

## Supported Platforms & Versions

| Platform | Supported Versions |
| :--- | :--- |
| Fabric | 1.21.11 |
